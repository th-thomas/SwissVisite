# Dossier technique
## Informations sur les différents fichiers
+ `consignes_GSB_PPE_3.pdf` : consignes pour le PPE 3 GSB (contexte, cahier des charges)
+ `existant\Swiss_Visite.accdb` : application Microsoft Access originale
+ `existant\BDDCompteRendusGSB.accdb` : fichier de base de données Microsoft Access original
+ `swissvisite_modele_conceptuel_de_donnees.Loo` : modèle conceptuel de données.
  + Pour ouvrir ce fichier, utiliser le logiciel [Looping](https://www.looping-mcd.fr/), développé par Patrick Bergougnoux.
+ `swissvisite_modele_conceptuel_de_donnees.png` : le même modèle conceptuel de données, exporté en tant qu'image.
+ `swissvisite_diagramme_de_classes.png` : diagramme de classes, réalisé à partir de Visual Studio 2019 et exporté en tant qu'image.
+ `swissvisite_migration_access_vers_mysql.pdf` : guide pour la migration des données depuis Access vers MySQL.
